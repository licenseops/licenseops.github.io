# LicenseOps Website

The documentation website for [LicenseOps](https://github.com/licenseops/licenseops) (`lops`), built with [Docusaurus](https://docusaurus.io/).

## Local Development

```bash
npm install
npm start
```

This starts a local development server at `http://localhost:3000`. Most changes are reflected live without restarting.

## Build

```bash
npm run build
```

Generates static files into the `build` directory.

## Deployment

Deployed automatically via GitHub Actions to GitHub Pages on push to `main`.
